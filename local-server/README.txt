ЛОКАЛЬНЫЙ АГЕНТ NUVIO
===================

Версия: 1.0.2

Инструкция по использованию:
---------------------------

1. Запустите файл local_agent.exe двойным кликом
2. В браузере автоматически откроется страница с токеном
3. Скопируйте токен и вставьте его в поле на странице Nuvio
4. Нажмите "Подключиться"

Важные замечания:
----------------

- Не закрывайте консольное окно агента во время работы с Nuvio
- При разрыве соединения перезапустите агент для получения нового токена
- Брандмауэр Windows может запросить разрешение на работу программы - разрешите доступ
- Агент работает локально на порту 8843

Решение проблем:
--------------

1. Если агент не запускается или не открывается браузер:
   - Проверьте, не заблокирован ли файл антивирусом
   - Попробуйте запустить от имени администратора

2. Если соединение прерывается:
   - Проверьте, не блокирует ли брандмауэр соединение
   - Убедитесь, что порт 8843 не используется другой программой

3. Если веб-приложение не может подключиться к агенту:
   - Проверьте правильность введенного токена
   - Перезапустите браузер и попробуйте снова
